The Pheneoum-Detection System is an advanced AI-based leaf analysis tool that detects plant phenotypes and diseases through deep learning models. The system extracts shape, texture, and color-based features, classifies leaf health, and provides farmers with accurate insights to take preventive actions. It is suitable for precision farming, crop monitoring, and agricultural research.

✔ Key Features

Automated plant phenotype detection

Leaf disease classification

Real-time prediction using camera or uploaded images

High accuracy CNN-based model

Easy-to-use interface for farmers and researchers

Scalable for multiple crops and environments

✔ Technologies

Python

TensorFlow/Keras

OpenCV

NumPy

Streamlit / Flask (UI)
